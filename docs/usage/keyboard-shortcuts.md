# Keyboard Shortcuts

## Global

| Shortcut | Action |
|----------|--------|
| **Cmd+P** | Open search |
| **Cmd+Shift+F** | Open search (alias) |
| **Cmd+1** | Focus editor |
| **Cmd+2** | Focus sidebar sections |
| **Escape** | Close search / git view |

## Git View

| Shortcut | Action |
|----------|--------|
| **Arrow Down/Up** | Navigate uncommitted files and commits |
| **Escape** | Exit git view |

## Search

| Shortcut | Action |
|----------|--------|
| **Arrow Down/Up** | Navigate results |
| **Enter** | Open selected result |
| **Escape** | Close search |

## Renaming

| Shortcut | Action |
|----------|--------|
| **Enter** | Confirm rename |
| **Escape** | Cancel rename |

## Editor

Standard CodeMirror keybindings apply:

| Shortcut | Action |
|----------|--------|
| **Cmd+Z** | Undo |
| **Cmd+Shift+Z** | Redo |
| **Cmd+A** | Select all |
| **Cmd+C/X/V** | Copy/Cut/Paste |
